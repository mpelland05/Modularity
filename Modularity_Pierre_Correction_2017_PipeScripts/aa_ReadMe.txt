Basic scripts contains all the scripts which are used to run the analyses. 

LaunchSpecificScripts contains all the script I (Maxime Pelland) use to launch the 
analyses such as selecting input for scripts, output folders and so on.

Model_LaunchScripts contains model scripts which can easily be used to launch the analyses. 

The text file AnalysisOrder tells you in which order to use the LaunchScripts.
